# Benchmark results

Generated by `python benchmarks/bench.py --write`. Deterministic (seeded synthetic data), so anyone reproduces these numbers.

**recall@k** = fraction of the true k nearest neighbors that the approximate index returns. **DB scanned** = share of the database whose distance was actually computed (HNSW's sublinear-search win).

Database: 4000 vectors × 64 dims (25 clusters), 200 queries, k=10, cosine. Build time: 44.60s.
